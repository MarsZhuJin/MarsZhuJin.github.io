Linux 命令 scp

Mar 11, 2017 | Hits

Linux scp命令用于Linux主机之间文件或者目录复制。
scp是 secure copy的缩写, scp是linux系统下基于ssh登陆进行安全的远程文件拷贝命令。

使用scp命令要确保用户具有远程文件的相应权限，否则无法起作用。

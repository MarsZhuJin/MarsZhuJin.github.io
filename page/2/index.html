use hexo create blog

本教程基于MacOS

1. 安装Git

​ 如果已经安装Xcode无需安装Git

2. 安装Node.js

​ Node.js 链接[http://pan.baidu.com/s/1hqHbx6s]

​ 或者是使用brew安装

3. 安装Hexo

参考官网

1. 安装软件
     1. sudo npm install hexo-cli -g
2. 创建
     1. mkdir blog
3. 初始化环境
     1. hexo init [or hexo init blog]
4. 安装依赖包
     1. npm install
5. 生成静态页面
     1. hexo generate [hexo g]
6. 启动本地调试
     1. hexo server [hexo s]
     2. 浏览器打开 http://localhost:4000/

4. 使用Git同步

  1. 注册Github账号
  2. 创建Repository
    1. 注意名称格式：用户名.github.io
    2. demo：zjd/zjd.github.io
  3. 生成SSH key
    1. 生成Key
      1. ssh-keygen -t rsa -C “zhu_303491@163.com”
    2. 添加SSH key到ssh-agent
      1. 在后台打开 ssh-agenteval”$(ssh-agent -s)”添加你的SSH Key到ssh-agentssh-add ~/.ssh/id_rsa
    3. 添加SSH key到Github
      1. 复制Key pbcopy < ~/.ssh/id_rsa.pub
      2. Github-> Setting->Add SSH key
    4. 测试
      1. ssh-T git@github.com

5. 部署本地文件到Github

  1. 编辑blog 根目录下_config.yml文件

    deploy:

    type: git

repository: http://github.com/zjd/zjd.github.io.git

branch: master

  2. 部署

    1. hexo g
    2. hexo d

  3. 如果遇到 deploy-git Error 则执行

    npm install hexo-deployer-git —save

hexo g

hexo d

6. 更换NexT主题

  1. 按照官网两种安装方式。我选择离线安装稳定版本 http://theme-next.iissnan.com/
  2. 将下载文件夹解压到blog/themes/目录下，并将文件夹更名为next
  3. 修改站点配置文件 /blog/_config.yml中themes 内容为next
  4. 重新部署查看效果

7. 主要命令

执行hero new “my new post”

在blog/source/_post中打开my_new_post.md 使用MarkDown语法写文章

写完后

hexo g 生成静态文件

hexo s 本地预览

hexo d git同步

使用 http://zjd.github.io 进行访问。
